## 📋 Información Técnica

### Meta Tags Implementados:

1. **Título de la página:**
   - `<title>`: Formulario de Inscripción - Tecnología en Desarrollo de software y programación

2. **Meta Description:**
   - Descripción para motores de búsqueda

3. **Open Graph (Facebook, WhatsApp, Telegram, LinkedIn):**
   - `og:type`: website
   - `og:title`: Formulario de Inscripción - Tecnología en Desarrollo de software y programación
   - `og:description`: Formulario de inscripción para el programa de Tecnología en Desarrollo de software y programación - SENECYT
   - `og:image`: URL de la imagen (1200x630px recomendado)
   - `og:url`: URL del sitio
   - `og:locale`: es_EC (Español Ecuador)
   - `og:site_name`: SENECYT

4. **Twitter Card:**
   - `twitter:card`: summary_large_image
   - `twitter:title`: Formulario de Inscripción - Tecnología en Desarrollo de software y programación
   - `twitter:description`: Formulario de inscripción para el programa de Tecnología en Desarrollo de software y programación - SENECYT
   - `twitter:image`: URL de la imagen

### 📸 Imágenes Recomendadas:

1. **og-image.jpg** (para Facebook, WhatsApp, LinkedIn):
   - Tamaño: 1200x630 píxeles
   - Formato: JPG o PNG
   - Ubicación: `src/assets/og-image.jpg`

2. **twitter-image.jpg** (para Twitter):
   - Tamaño: 1200x630 píxeles (o 1200x675 para mejor visualización)
   - Formato: JPG o PNG
   - Ubicación: `src/assets/twitter-image.jpg`

### ⚠️ Nota Importante:

**Debes actualizar las URLs en `index.html`** con la URL real de tu sitio cuando esté en producción:
- Cambiar `https://registro-senecyt.com/` por tu dominio real
- Asegurarte de que las imágenes estén accesibles públicamente
- Las imágenes deben tener rutas absolutas (con https://)

### 🔍 Herramientas para Probar:

1. **Facebook Debugger:** https://developers.facebook.com/tools/debug/
2. **Twitter Card Validator:** https://cards-dev.twitter.com/validator
3. **LinkedIn Post Inspector:** https://www.linkedin.com/post-inspector/
